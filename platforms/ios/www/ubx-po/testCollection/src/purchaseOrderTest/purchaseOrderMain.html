Welcome to the Ubiquity XForms purchasing system demo. This form is designed to show you some of the benefits of using XForms as a client side forms mark-up over standard HTML. The goal of this demo is to allow you to see how the XForms processor and code written can help to make your life easier.

Questions regarding this form or the markup can be directed towards the Ubiquity XForms Developer group. Prioir to posting make sure you have followed the Ubiquity XForms Purchase Order Tutorial and consulted the Xforms 1.1 spec.

Quick Start

For the people who want nothing to do with following the Tutorial here is a breif overview of what needs to happen to properly run this application:

  • 1. Dowload eXist database, yes this uses an XML database. You will also need a JDK for eXist.
  • 2. "Dowload" Ubiquity XForms from here
  • 3. Since you already have these files I'm assuming you have installed Ubiquity and eXist.
  • 4. Check out the cross site scripting discussion if you run into problems
  • 5. usr: admin pw: test

There you go have some fun and play around with Ubiquity.
